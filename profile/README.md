# ownCloud Community

This organisation is a **recognition space** for everyone who contributed to ownCloud.

If you are a member here, it means your work helped build ownCloud. That contribution is part
of the project's history and we want it to remain visible on your GitHub profile.

## What this org is

- A permanent record of ownCloud community membership
- A place that stays on your GitHub profile and CV
- Recognition of your past contribution to one of the largest open source file-sync projects

## What this org is not

- This org does not host active repositories
- Membership here is not a demotion — it is acknowledgement

## Coming back

If you want to contribute again, the door is open. Open a PR, join a discussion, or start
reviewing. The contributor ladder is published in the
[ownCloud Governance Charter](https://github.com/owncloud/.github).

Active contributors, maintainers, and reviewers work in the main
[github.com/owncloud](https://github.com/owncloud) organisation.
